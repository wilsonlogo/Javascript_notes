API Extraction

In XMLHttpRequest, readyState represents the current state of the request. It shows how far along the request is in the process of communicating with the server.



    Value  State	                Meaning
    0	   UNSENT	            Request created, but not opened yet (xhr.open() not called)
    1	   OPENED	            xhr.open() has been called
    2	   HEADERS_RECEIVED     xhr.send() has been called, and headers have been received
    3	   LOADING	            Response is being received (browser is downloading the data)
    4	   DONE	                Request finished â€” response is fully received
